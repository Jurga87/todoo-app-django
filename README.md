# Django Practice Project

This is a simple practice web application built with the **Django** framework.  
The project follows the tutorial series from this YouTube playlist:  

https://www.youtube.com/watch?v=hWhWdf2gsfQ&list=PL_SWATNcLmYmlMmEIb18iZldUEwAOONke

The goal of the project is to learn the basics of Django, understand its structure, and practice building a small web application with user authentication and task management.
